kameloso

Functionality related to connecting to a server over the Internet.

Includes core.thread.fiber.Fibers that help with resolving the address of, connecting to, and reading full string lines from a server.

Having them as core.thread.fiber.Fibers means a program can do address resolution, connecting and reading while retaining the ability to do other stuff concurrently. This means you can conveniently run code inbetween each connection attempt, for instance, without breaking the program's flow.
