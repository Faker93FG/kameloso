highlightEmotesImpl

kameloso plugins printer formatting

Highlights Twitch emotes in the chat by tinting them a different colour, saving the results into a passed output range sink.

version(WithPlugins && WithPrinterPlugin && Colours && TwitchSupport)

Parameters

line string

Content line whose containing emotes should be highlit.

sink Sink

Output range to put the results into.

emotes string

The list of emotes and their positions as divined from the IRCv3 tags of an event.

pre TerminalForeground

Terminal foreground tint to colour the emotes with.

post TerminalForeground

Terminal foreground tint to reset to after colouring an emote.

colourful Flag!"colourful"

Whether or not emotes should be highlit in colours.

brightTerminal Flag!"brightTerminal"

Whether or not the terminal has a bright background and colours should be adapted to suit.
