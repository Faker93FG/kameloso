nameOf

kameloso plugins common base

Returns either the nickname or the display name of a user, depending on whether the display name is known or not. Overload that looks up the passed nickname in the passed plugin's users associative array of dialect.defs.IRCUsers.

If not version TwitchSupport then it always returns the nickname.
