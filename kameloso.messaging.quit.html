Parameters

priority

Whether or not to send the message as a priority message, received before other messages are, if there are several. Default to Yes.priority, since we're quitting.

state IRCPluginState

The current plugin's kameloso.plugins.common.core.IRCPluginState, via which to send messages to the server.

reason string

Optionally, the reason for quitting.

quiet Flag!"quiet"

Whether or not to echo what was sent to the local terminal.
