LogLineBuffer

kameloso plugins printer logging

A struct containing lines to write to a log file when next committing such.

This is only relevant if kameloso.plugins.printer.base.PrinterSettings.bufferedWrites is set.

As a micro-optimisation an std.array.Appender is used to store the lines, instead of a normal string[].

version(WithPlugins && WithPrinterPlugin)
package
struct LogLineBuffer {
string dir;
string file;
Appender!(string[]) lines;
this(string dir, SysTime now);
