kameloso.plugins.pipeline

kameloso plugins

The Pipeline plugin opens a Posix named pipe in the current directory, to which you can pipe text and have it be sent verbatim to the server.

It has no commands; indeed, it doesn't listen to dialect.defs.IRCEvents at all, only to what is sent to it via the named FIFO pipe.

This requires version Posix, which is true for UNIX-like systems (like Linux and macOS).
