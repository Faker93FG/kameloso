queryTwitchImpl

kameloso plugins twitchbot api

Sends a HTTP GET request to the passed URL, and "returns" the response by adding it to the shared bucket associative array.

Callers can as such spawn this function as a new thread and asynchronously monitor the bucket for when the results arrive.

version(WithPlugins && TwitchSupport && WithTwitchBotPlugin && TwitchAPIFeatures)

Parameters

url string

URL address to look up.

authToken string

Authorisation token HTTP header to pass.

timeout uint

How long to let the query run before timing out.

bucket QueryResponse[string]

The shared associative array to put the results in, response values keyed by URL.

caBundleFile string

Path to a cacert.pem SSL certificate bundle.

Examples

immutable url = "https://api.twitch.tv/helix/some/api/url";

spawn&(&queryTwitchImpl, url, plugin.authorizationBearer,
    plugin.queryResponseTimeout, plugin.bucket, caBundleFile);
delay(plugin, plugin.approximateQueryTime, Yes.msecs, Yes.yield);
immutable response = waitForQueryResponse(plugin, url);
// response.str is the response body
