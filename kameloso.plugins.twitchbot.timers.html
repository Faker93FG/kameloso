kameloso.plugins.twitchbot.timers

kameloso plugins twitchbot

Implementation of Twitch bot timers. For internal use.

The dialect.defs.IRCEvent-annotated handlers must be in the same module as the kameloso.plugins.twitchbot.TwitchBotPlugin, but these implementation functions can be offloaded here to limit module size a bit.
