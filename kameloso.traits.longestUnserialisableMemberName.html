longestUnserialisableMemberName

kameloso traits

Gets the name of the longest member in one or more aggregate types, including lu.uda.Unserialisable ones.

This is used for formatting terminal output of objects, so that columns line up.

alias longestUnserialisableMemberName(Things...) = longestMemberNameImpl!(Yes.all, Things)

Parameters

Things

Types to introspect and count member name lengths of.

Examples

1 import lu.uda : Hidden, Unserialisable;
2 
3 struct Foo
4 {
5     string veryLongName;
6     int i;
7     @Unserialisable string veryVeryVeryLongNameThatIsValidNow;
8     @Hidden float likewiseWayLongerButInvalidddddddddddddddddddddddddddddd;
9     deprecated bool alsoVeryLongButDeprecated;
10     void aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa();
11     void bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb(T)();
12 }
13 
14 struct Bar
15 {
16     string evenLongerName;
17     float f;
18 
19     @Unserialisable
20     @Hidden
21     long looooooooooooooooooooooong;
22 }
23 
24 static assert(longestUnserialisableMemberName!Foo == "veryVeryVeryLongNameThatIsValidNow");

26 static assert(longestUnserialisableMemberName!(Foo, Bar) == "veryVeryVeryLongNameThatIsValidNow");
