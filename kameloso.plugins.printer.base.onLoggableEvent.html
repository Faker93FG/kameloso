onLoggableEvent

kameloso plugins printer base

Logs an event to disk.

It is set to kameloso.plugins.common.core.ChannelPolicy.any, and configuration dictates whether or not non-home events should be logged. Likewise whether or not raw events should be logged.

Lines will either be saved immediately to disk, opening a std.stdio.File with appending privileges for each event as they occur, or buffered by populating arrays of lines to be written in bulk, once in a while.
