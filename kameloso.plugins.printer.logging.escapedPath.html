escapedPath

kameloso plugins printer logging

Replaces some characters in a string that don't translate well to paths.

This is platform-specific, as Windows uses backslashes as directory separators and percentages for environment variables, whereas Posix uses forward slashes and dollar signs.

Parameters

path string

A filesystem path in string form.

Return Value

Type: auto

The passed path with some characters replaced.

Bugs

Escaped paths can collide with real files named what the original path was escaped to. "%PATH%" may as such collide with "_PATH_", as the former was escaped to an already valid filename.

2     immutable before = escapedPath("unchanged");
3     immutable after = "unchanged";
4     assert((before == after), after);
