queryTwitch

kameloso plugins twitchbot api

Wraps queryTwitchImpl by either starting it in a subthread, or by calling it normally.

Once the query returns, the response body is checked to see whether or not an error occurred. If so, it throws an exception with a descriptive message.

Note: Must be called from inside a core.thread.fiber.Fiber.
