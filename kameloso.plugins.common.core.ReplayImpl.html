ReplayImpl

kameloso plugins common core

Implementation of the notion of a function call with a bundled payload dialect.defs.IRCEvent, used to replay a previous event.

It functions like a Command pattern object in that it stores a payload and a function pointer, which we queue and issue a WHOIS query. When the response returns we trigger the object and the original dialect.defs.IRCEvent is replayed.
