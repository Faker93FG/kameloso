kameloso plugins seen SeenPlugin

This mixes in functions that fully implement an kameloso.plugins.common.core.IRCPlugin. They don't do much by themselves other than call the module's functions.

As an exception, it mixes in the bits needed to automatically call functions based on their dialect.defs.IRCEvent.Type annotations. It is mandatory if you want things to work, unless you're making a separate implementation yourself.

Seen from any other module, this module is a big block of private things they can't see, plus this visible plugin class. By having this class pass on things to the private functions we limit the surface area of the plugin to be really small.
