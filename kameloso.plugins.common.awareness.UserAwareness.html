UserAwareness

kameloso plugins common awareness

Implements *user awareness* in a plugin module.

Plugins that deal with users in any form will need event handlers to handle people joining and leaving channels, disconnecting from the server, and other events related to user details (including services account names).

If more elaborate ones are needed, additional functions can be written and, where applicable, annotated appropriately.

ChannelPolicy channelPolicy = ChannelPolicy.home

) {
