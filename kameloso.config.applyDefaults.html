applyDefaults

kameloso config

Completes a client's, server's and bot's member fields. Empty members are given values from compile-time defaults.

Nickname, user, GECOS/"real name", server address and server port are required. If there is no nickname, generate a random one. For any other empty values, update them with relevant such from kameloso.constants.KamelosoDefaults (and kameloso.constants.KamelosoDefaultIntegers).

Examples

1 import kameloso.constants : KamelosoDefaults, KamelosoDefaultIntegers;
2 import std.conv : text;
3 
4 IRCClient client;
5 IRCServer server;
6 IRCBot bot;
7 
8 assert(!client.nickname.length, client.nickname);
9 assert(!client.user.length, client.user);
10 assert(!client.ident.length, client.ident);
11 assert(!client.realName.length, client.realName);
12 assert(!server.address, server.address);
13 assert((server.port == 0), server.port.text);
14 
15 applyDefaults(client, server, bot);
16 
17 assert(client.nickname.length);
18 assert((client.user == KamelosoDefaults.user), client.user);
19 assert(!client.ident.length, client.ident);
20 assert((client.realName == KamelosoDefaults.realName), client.realName);
21 assert((server.address == KamelosoDefaults.serverAddress), server.address);
22 assert((server.port == KamelosoDefaultIntegers.port), server.port.text);
23 assert((bot.quitReason == KamelosoDefaults.quitReason), bot.quitReason);
24 assert((bot.partReason == KamelosoDefaults.partReason), bot.partReason);
25 
26 client.nickname = string.init;
27 applyDefaults(client, server, bot);
28 
29 assert(client.nickname.length, client.nickname);
