resolveFiber

kameloso net

Given an address and a port, resolves these and populates the array of unique std.socket.Address IPs inside the passed Connection.

@safe
@system in (!conn.connected, "Tried to set up a resolving fiber on an already live connection")

ref Connection conn

const ushort port
,
const bool useIPv6

Parameters

conn Connection

Reference to the current Connection.

address string

String address to look up.

port ushort

Remote port build into the std.socket.Address.

useIPv6 bool

Whether to include resolved IPv6 addresses or not.

abort bool

Reference "abort" flag, which -- if set -- should make the function return and the core.thread.fiber.Fiber terminate.

1 import std.concurrency : Generator;

3 Connection conn;

6 auto resolver = new Generator!ResolveAttempt(() =>

11 resolveloop:

18     {

22     case success:
23         // Address was resolved, the passed `conn` was modified
24         break resolveloop;
25 
26     case exception:
27         // Recoverable

30 
31     case failure:
32         // Resolution failed without errors
33         failGracefully(attempt.error);
