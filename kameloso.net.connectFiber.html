connectFiber

kameloso net

Fiber function that tries to connect to IPs in the ips array of the passed Connection, yielding at certain points throughout the process to let the calling function do stuff inbetween connection attempts.

@safe
@system in (!conn.connected, "Tried to set up a connecting fiber on an already live connection")
void
connectFiber
(
ref Connection conn
,
const uint connectionRetries
,
ref bool abort
)

Parameters

conn Connection

Reference to the current, unconnected Connection.

connectionRetries uint

How many times to attempt to connect before signaling that we should move on to the next IP.

abort bool

Reference "abort" flag, which -- if set -- should make the function return and the core.thread.fiber.Fiber terminate.
