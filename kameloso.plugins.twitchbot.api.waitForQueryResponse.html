waitForQueryResponse

kameloso plugins twitchbot api

Common code to wait for a query response.

Merely spins and monitors the shared bucket associative array for when a response has arrived, and then returns it.

Times out after a hardcoded kameloso.plugins.twitchbot.base.TwitchBotPlugin.queryResponseTimeout

if nothing was received.

Note: Must be called from inside a core.thread.fiber.Fiber.
