applyCustomSettings

kameloso plugins common base

Changes a setting of a plugin, given both the names of the plugin and the setting, in string form.

This merely iterates the passed plugins and calls their setSettingByName methods.

bool
applyCustomSettings
(
IRCPlugin[] plugins
,
const string[] customSettings
,
CoreSettings copyOfSettings
)

Parameters

plugins IRCPlugin[]

Array of all kameloso.plugins.common.core.IRCPlugins.

customSettings string[]

Array of custom settings to apply to plugins' own setting, in the string forms of "plugin.setting=value".

copyOfSettings CoreSettings

A copy of the program-wide kameloso.kameloso.CoreSettings.

Return Value

Type: bool

true if no setting name mismatches occurred, false if it did.
