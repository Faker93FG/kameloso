Repeater

kameloso plugins common mixins

Implements queueing of events to repeat.

This allows us to deal with triggers both in dialect.defs.IRCEvent.Type.RPL_WHOISACCOUNT and dialect.defs.IRCEvent.Type.ERR_UNKNOWNCOMMAND while keeping the code in one place.
