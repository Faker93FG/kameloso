Connection.sendline

kameloso net Connection

Sends a line to the server.

Intended for servers that deliminates lines by linebreaks, such as IRC servers.

struct Connection
void
sendline
(
uint maxLineLength = 512
Data...
)
(
const Data data

Parameters

maxLineLength

Maximum line length before the sent message will be truncated.

data Data

Variadic list of strings or characters to send. May contain complete substrings separated by newline characters.

Examples

conn.sendline("NICK foobar");
conn.sendline("PRIVMSG #channel :text");
conn.sendline("PRIVMSG " ~ channel ~ " :" ~ content);
conn.sendline("PRIVMSG ", channel, " :", content);  // Identical to above
conn.sendline!1024(longerLine);  // Now with custom line lengths
