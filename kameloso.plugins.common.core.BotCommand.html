BotCommand

kameloso plugins common core

Defines an IRC bot command, for people to trigger with messages.

If no PrefixPolicy is specified then it will default to PrefixPolicy.prefixed and look for kameloso.kameloso.CoreSettings.prefix at the beginning of messages, to prefix the command word. (Usually "!", making it "!command".)

struct BotCommand {
PrefixPolicy policy;
string word;
bool hidden;
this(PrefixPolicy policy, string word, Flag!"hidden" hidden);
this(string word);
}
